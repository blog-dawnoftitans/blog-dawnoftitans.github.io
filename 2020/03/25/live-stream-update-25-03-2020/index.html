Divine Shrine Update: 

The 5-STAR Golden Relics are finally here!

We’ve made a number of changes to the Divine Altars. Below is a summary of what the new Altars contain, and the drop rates of the Relics within them:

  • GOLDEN RELICS – 7,000 DGs
    • Contains 5* Golden Set of Divine Relics
    • Drop Rates: 
      • 5-STAR: 100%

  • NORSE RELICS – 5,000 DGs
    • Contains 5* Norse Set of Divine Relics
    • Drop Rates: 
      • 5-STAR: 100%

  • DIVINE RELICS – 2,000 DGs
    • Contains 5* Norse Set and 5* Original Set of Divine Relics
    • Drop Rates:
      • 5-STAR (Original Set): 90%
      • 5-STAR (Norse Set): 10%

  • BLESSED RELICS – 1,000 DGs
    • Contains powerful 4* Relics (e.g. Banners, Runes, Orbs)
    • Contains 5* Original Set of Divine Relics
    • Drop Rates:
      • 4-STAR: 75%
      • 5-STAR (Original Set): 25%

We will be making further changes to the Divine Shrine in the coming weeks, and as always, we welcome your feedback on these.
